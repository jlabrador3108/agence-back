# Test Agence Back

## Technologies

- Nestjs
- Prisma
- MySQL

## Installation and configuration

1. **Clone repository**

   ```bash
   git clone <REPOSITORY_URL>
   cd <PROJECT_NAME>

   ```

2. **Install dependencies**

   ```bash
   yarn install

   ```

3. **Run**

   ```bash
    yarn start:dev
   ```

4. **Documentation Swagger**

- http://localhost:5000/docs
